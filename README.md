# 🎵 音樂專業個人英文字典 App (English Dictionary App)

這是一款專為音樂愛好者與樂手設計的極簡英文字典學習 App。

## 🌟 核心特色
- **專業音樂術語庫**：內建超過 80 筆音樂專業領域單字，包含白話解析與音樂情境例句。
- **每日學習系統**：每日自動生成學習進度，結合 SRS 間隔重複記憶法。
- **聽力練習模式**：智慧生成的聽辨測驗，支援連續答對計數 (Streak)。
- **專業發音支援**：所有單字均支援高品質 TTS 英文發音。
- **單手優化操作**：支援左右滑動學習卡片與大按鈕設計，適合行動裝置使用。

## 🛠️ 技術架構
- **Frontend**: Vanilla JS (純 JavaScript), HTML5, CSS3
- **Design System**: 極簡主義、高對比度現代風格
- **Storage**: 使用 LocalStorage 進行本地資料持久化，無需註冊登入。

## 🚀 如何啟動
直接開啟 `index.html` 即可在瀏覽器中執行。

## 📄 授權規範
本專案僅供個人學習與開發參考使用。
